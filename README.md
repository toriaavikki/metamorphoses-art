# Метаморфозы
Интерактивный арт-объект о психоделических измерениях сознания.

## Описание
Сайт-путешествие через метаморфозы мышления, где каждый клик — попытка сдвига, а каждое взаимодействие — шаг в лабиринт собственных мыслей.

## Технологии
- Vanilla JavaScript
- WebGL шейдеры
- Web Audio API
- CSS анимации

## Структура
- `/` — Лендинг (Манифест статики)
- `/archive.html` — Архив обломков
- `/journey-*.html` — Отдельные путешествия

## Разработка
1. Клонируйте репозиторий
2. Откройте index.html в браузере
3. Или запустите локальный сервер: `npx serve`
